# 更完整系统（GWZOS）

更完整系统的目的是编写一个异步功能完整的异步操作系统。我们希望尽可能完整地实现异步内核的核心概念，提供相应的驱动、软件生态系统，为未来内核的设计探索可能的实现方案。对比不同解决方案的性能开销，得到较详细的内核验证结论。
